# AI-Fashion-Visual-Search
AI-powered fashion recommendation system that uses ResNet50 CNN embeddings and FAISS similarity search to find visually similar clothing items from uploaded images. Built with Python, TensorFlow, FAISS, and Streamlit.
## Features
- Image upload functionality
- CNN-based feature extraction using ResNet50
- Fast similarity search using FAISS
- Interactive web interface with Streamlit
- Real-time fashion product recommendations

## Tech Stack
- Python
- TensorFlow / Keras
- ResNet50
- FAISS
- Streamlit
- NumPy

## Workflow
1. User uploads a clothing image.
2. ResNet50 extracts image embeddings.
3. FAISS performs similarity search on the product database.
4. The system returns the most visually similar clothing items.
